From form to live card in 6 steps

No servers, no database. Edit here, download a file, push it to GitHub, then write the tag and print the QR. Everything auto-saves in this browser as you type.

1
Fill in the client
Pick a client (or hit New client), edit the Card content tab, toggle off anything they don't have. Watch the phone preview.
2
Download the card
Hit Download index.html. That single file is the finished website.
3
Push to GitHub
Drop it in a repo named index.html, turn on GitHub Pages in Settings → Pages. You get a public URL.
4
Write the NFC tag once
Open NFC Tools → Write → URL record → paste the Pages link → hold the NTAG213 to your phone. Then set it read-only.
5
Generate the QR
Open the QR code tab, paste the same URL, download PNG or SVG, and print it.
6
Update anytime
Client changed a number? Edit here, re-download, re-commit. Same URL, so the tag and QR keep working forever.
